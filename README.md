# hey, i'm aaryan 👋

I'm a second-year Computer Science student specializing in **Artificial Intelligence & Machine Learning**.

Most of my time is spent building Python libraries, learning Data Science, and automating things because repeating the same task twice already feels illegal.

Currently, I'm spending an unreasonable amount of time teaching computers to understand data while simultaneously convincing myself that my code will work on the first run. (It never does.)

Sometimes I build cool stuff.
Sometimes I spend three hours debugging a missing comma.
Both count as progress.
